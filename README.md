# ZeroTrust-App


# 🌐 Zero Trust Dashboard Web App

A simple and interactive **Zero Trust Security Dashboard** built using HTML, CSS, and JavaScript. This project simulates device monitoring and threat detection in a clean UI.

---

## 🚀 Features

* Displays total number of devices
* Simulates real-time threat alerts
* Clean and responsive UI design
* Interactive button to generate threats
* Lightweight and easy to run

---

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript

---

## 📂 Project Structure

ZeroTrust-App/
│── index.html
│── style.css
│── app.js

---

## ▶️ How to Run

1. Download or clone this repository
2. Open the project folder in VS Code
3. Right-click index.html
4. Click "Open with Live Server"

The web app will open in your browser

## 🔐 About Zero Trust

Zero Trust is a modern security model based on the principle:
**“Never trust, always verify”** — meaning every user and device must be continuously authenticated before access is granted. ([DEV Community][1])
